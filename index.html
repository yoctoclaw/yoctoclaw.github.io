You can read the entire codebase in an hour.

agent.zig    — core loop (250 lines)
api.zig      — multi-provider HTTP (329 lines)
stream.zig   — SSE parser (344 lines)
json.zig     — hand-rolled JSON (500 lines)
tools_coding.zig — 7 tools (280 lines)
context.zig  — token management (225 lines)

Works with Claude, OpenAI, and Ollama. Switch providers with a flag.

Three profiles, one codebase

Compile-time profiles swap entire tool sets. Only the selected profile ships — zero runtime overhead.

coding     — bash, read/write/edit files, search, list, apply_patch
iot        — MQTT pub/sub, HTTP requests, key-value store
robotics   — robot commands, e-stop, telemetry, bounds checking

Build with -Dprofile=iot or -Dsandbox=true for restricted execution.
