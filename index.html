Three profiles, one codebase

Compile-time profiles swap entire tool sets. Only the selected profile ships ‚Äî zero runtime overhead.

coding     ‚Äî bash, read/write/edit files, search, list, apply_patch
iot        ‚Äî MQTT pub/sub, HTTP requests, key-value store
robotics   ‚Äî robot commands, e-stop, telemetry, bounds checking

Build with -Dprofile=iot or -Dsandbox=true for restricted execution.
